# 🏥 Patient Diagnostic Analytics Dashboard (Power BI)

## 📌 Overview

This project presents an interactive Power BI dashboard designed to analyze patient diagnostic data, identify high-risk individuals, and monitor key health indicators.

## 🎯 Objectives

* Analyze patient risk levels
* Monitor chronic conditions (Diabetes, Hypertension, Inflammation)
* Identify disease patterns and trends
* Provide actionable healthcare insights

## 📊 Dashboard Pages

### 1. Executive Overview

* Total Patients & Visits
* Risk Distribution
* Fever Type Analysis
* Trends over time

### 2. Health Monitoring

* Diabetes & Blood Pressure Analysis
* Platelet Risk & CRP Trends
* Chronic condition indicators

### 3. Patient Insights

* Risk segmentation
* Age group analysis
* Disease vs risk relationships

## 🛠 Tools Used

* Power BI
* Power Query
* DAX

## 📈 Key Insights

* High-risk patients are concentrated in specific age groups
* Certain fever types correlate with severe conditions
* Chronic indicators such as diabetes and hypertension increase patient risk

## ⚠ Disclaimer

This dashboard is generated using simulated data and AI-assisted analysis. Results may not reflect real-world clinical accuracy.
